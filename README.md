# neurality-dev/.github

This repository hosts the Neurality GitHub organization profile README and any shared org-level configuration (issue templates, workflows) we later choose to standardize.

The profile README visible at [github.com/neurality-dev](https://github.com/neurality-dev) is sourced from [`profile/README.md`](profile/README.md).

## Scope

- `profile/README.md` - public-facing org profile.
- Reserved for future shared assets (issue templates, community health files).

## Why this repo is public

GitHub renders the org profile README only when this repository is public, or to org members when it is private. Since the goal is for visitors to learn about Neurality, this repo is public. Source code for Neurality products lives in separate private repositories.
